# 🐍 Snake

## Project Roadmap

### 🟢 Current Functionality

- **Core Mechanics:** Snake can move, eat, and grow.
- **Spawning:** Food spawns randomly in empty spaces.
- **UI:** Basic text rendering and score tracking.
- **Balance:** FPS is locked to **15** for easier maneuverability.

---

### 🟡 Current Bugs

- [ ] **Text Scaling:** No way to control rendered text width and height.
- [ ] **Viewport Issues:** Improper viewport handling causing UI/gameplay glitches.
- [ ] **Edge Ghosting:** Snake parts occasionally render on the opposite edge when near boundaries.

---

### 🔵 Future Upgrades

- [ ] **Menu System:** Include settings to adjust snake speed.
- [ ] **Pause Functionality:** Ability to freeze the game mid-session.
- [ ] **Fullscreen Support:** Toggle between windowed and fullscreen modes.
- [ ] **Save System:** Persistent high scores stored in a file with player names.
- [ ] **Cool Textures:** Replace flat colors with detailed sprites.
- [ ] **Animations:** Smoother transitions for movement and growth.
- [ ] **Camera Shake:** Add screen-shake effects when eating food.
- [ ] **Procedural Levels:** Generate dynamic maps/obstacles.
- [ ] **Smart Spawning:** Change food respawn from random tiles to an **empty-tile-based search**.
- [ ] **Special Foods:** Add variety (e.g., speed boosts, double points).
- [ ] **Sound System:** Add SFX for eating, dying, and clicking menus.
